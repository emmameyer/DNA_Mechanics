[naming_commands.txt](naming_commands.txt): list of useful commands for pymol edits to DNA: **IMPOTANT NOTE** when exporting a molecule in pymol, if you have not typed "sort," it will do so for you. If your objects are not labelled separately and overlap, IE two residues named DA and also named residue 1, it will attempt to sort this and mess up the system. Then when you go to select residue 1, it will select BOTH of them, which is an issue. This usually comes up when you are combining/ligating objects, like two pieces of DNA. It's especially important to renumber your DNA when you ligate two pieces (a semicircular piece attached to a linear, etc.), so that there are not duplicate residue numbers or it **will** get messed up.
